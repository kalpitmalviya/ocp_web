# ocp_web
1. Create a new application on OpenShift:
    ```bash
    oc new-app https://github.com/kalpitmalviya/ocp_web.git
    ```

2. Start the build process:
    ```bash
    oc start-build ocpweb
    ```

This will deploy your application on OpenShift.
